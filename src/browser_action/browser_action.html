Dev Tabs

To change your settings, go to the 'settings' page by clicking the cog in the middle of the page, under the weather info. Also for more info on this extension visit the Github page, or check my website at mattaz.com

Responsive View

This is for checking a website in different viewports. Enter the url of the website you would like to see in multiple viewport sizes.

QR Code Generator

Copy a url and paste it into the input box. Press 'Get QR Code' and it will generate a QR code which you can scan with your phone and it will bring you to the page you were on. Or if you just want to create some QR codes for other things, this will work.
